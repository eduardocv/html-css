Abrindo links em novas guias

O parâmetro target vai definir o comportamento do link. Entre os possíveis valores temos:

  • _blank: vai abrir o link em uma nova janela
  • _self: vai abrir na janela atual (padrão)
  • _top: vai desfazer todos os frames e abrir no navegador completo
  • _parent: similar ao _top
  • nome-do-frame: caso esteja usando frames, indicar o nome da janela a abrir

Esse link é seu?

Podemos indicar a natureza do destino do link utilizando o parâmetro rel que aceita os seguintes valores

  • next: indica que o link é a próxima parte do documento atual.
  • prev: indica que o link é a parte anterior do documento atual.
  • author: indica que é o link para o site do autor do artigo atual.
  • external: indica que é um link para outro site que nõ faz parte do site atual.
  • nofollow: indica que é um link para um site não endossado, como um link pago.
